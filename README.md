# YTDExtreme

Cross platform YouTube video downloading software built in ElectronJS <br>
Powered By: https://github.com/fent/node-ytdl-core

## Requirements

- NodeJS
- ElectronJS

## Instructions to run

### Step 1

Install Dependencies

```bash
$ npm install
```

### Step 2

Run Project

```bash
$ npm start
```

## Instructions to build

### Windows

```bash
$ npm run package-win
```

### Linux

```bash
$ npm run package-linux
```

### Mac

```bash
$ npm run package-mac
```
